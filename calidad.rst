Testing
-------

Frameworks para la automatización de pruebas unitarias.:

	* Django - `Django Testing <https://docs.djangoproject.com/en/1.9/topics/testing/>`_
	* JavaScript - `QUnit <https://qunitjs.com/>`_

Existen otros frameworks que te permiten calcular el "coverage" (cobertura de código testeado).

Analizadores de código
----------------------

Frameworks para analizar el código y conocer la deuda técnica del proyecto.:

	* Básico - `JSHint <http://jshint.com/>`_. Solamente para JavaScript.
	* Avanzado - `SonarQube <http://sonarqube.org/>`_: `Ejemplo <http://nemo.sonarqube.org/>`_
	* CSS - `CssLint <http://csslint.net/>`_. Solamente para CSS.

Code Style Checker
------------------

Frameworks para verificar la correcta estilización del código.:

	* JavaScript - `JSCS <http://jscs.info/>`_. Permite relacionarse con Sublime y Git.
